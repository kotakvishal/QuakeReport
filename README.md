# QuakeReport
---
## Descripton
This app helps us to get a information about ten most recent earthquakes all over the world. User can also filter the information with the magnitude of the earthquakes.

## Technology used
* Java
* XML
* Android Studio(IDE)

## Working
* The home page of the app displays the information about most recent earthquakes all over the world.
* This information includes, the magnitude of earthquake, the place where it is originated along with the date and time.
* There is also a setting preference, where a user can modify or filter the outcomes based on the magnitude.

## Built for
Android phones and tablets(Jelly Bean or above).
